###Temática
Este proyecto surge como la necesidad de la empresa La Gloria S.L. de tener una página web acorde a las tecnologías actuales. La página web de dicha empresa
está hecha en Flash, por lo que es bastante antigua y requiere muchas mejoras. 

###Tecnologías
Para cumplir con los requisitos básicos solicitados por La Gloria S.L. se desarrollará una página web con las siguientes tecnologías: 

* <b>MongoDB</b>: como base de datos de la aplicación.
* <b>NodeJS</b>: para la implementación del servidor.
* <b>AngularJS</b>: para el cliente.
* <b>Jade</b>: para la simplificación de las plantillas HTML.

###Casos de uso
Muchas acciones de la web se podrán realizar sin necesidad de iniciar sesión con un usuario previamente creado. Entre estas acciones se encuentran:

* Ver la información de contacto de La Gloria S.L.
* Ver todos los productos de la gama La Gloria S.L.
* Ver todos los productos de la gama Publicitarios.
* Añadir productos a un carrito de la compra.

Sin embargo, para ciertas funcionalidades necesitaremos tener un usuario. Se necesitará para:

* Realizar un pedido.
